# Gobuster commands used so far

## Commands

- `gobuster dir -u <url> -w <path to wordlist> -x <extension>` to find directory
- `-U <username>`
- `-s <statusCodes>` whitelisted statuscodes
- `-k` No ssl/tls certificate varification
- `-H` specify http headers
